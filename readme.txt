== Description ==

Hides text behind a black text/black background tile. Holding the mouse over the text reveals it. Insipred by Something Awful's spoiler system.

Usage: Simply put a [SPOILER]tag around your text[/SPOILER] that you wish to become "spoilered." Works in both comments and posts.

<a href="http://www.thedailyblitz.org/saspoiler-a-wordpress-plugin">See a sample of it in use here.</a>

== Installation ==

1. Upload `saspoiler.php` or it's containing directory to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
